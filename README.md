# hooks-snake-game
サイト: https://codesandbox.io/s/github/TakumaSatake/hooks-snake-game

## 機能
- ゲーム画面	
    - タイトルコンポーネント
    - ナビゲーションコンポーネント
    - フィールドコンポーネント
    - ボタンコンポーネント
    - 操作ボタンコンポーネント

## 技術構成
- react 17.0.2
- react-dom 17.0.2
- react-scripts 4.0.0

## 開発方法

```bash
# パッケージをインストール
$ yarn install

# 静的生成したアプリケーションを起動
$ yarn start
```
